C++ programming(in 2022)
C++ (pronounced "C plus plus") is a general-purpose programming language that was developed as an extension of the C programming language. It was created by Bjarne Stroustrup in the late 1970s and first appeared in 1985. C++ is known for its ability to combine high-level features with low-level memory manipulation, making it suitable for a wide range of applications.
